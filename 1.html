HTML의 기본구조

<!doctype html> <html> <head> <title> 문서 제목 </title> </head> <body> <h1> 제목 </h1> <p> 내용 </p> </body> </html>
  • <!DOCTYPE> : 현재 문서의 HTML 타입을 정의하며, HTML5 의 경우 <!DOCTYPE html> 로 명시한다.
  • <html> : HTML 문서의 시작과 끝을 표시한다.
  • <head> : 웹페이지 문서에 대한 정보로 메타데이터(metadata)를 정의한다.
    • style, meta, link, script, base 태그 등으로 정의된다.
    • <title> : HTML 문서의 제목을 정의한다.
  • <body> : 내부에 웹 페이지의 내용을 작성한다.
    • <h1> ~ <h6> : 본문 내용의 제목을 크기별로 나타낸다.
    • <p> : 본문의 각 단락을 구분한다.

앵커 태그

  • <a href="주소" target=_내용> 표시할 검색어 <a> : 하이퍼링크를 달아서 다른 주소 연결할 때 사용.
  • <a href="#주소 혹은 위치/id 등"> 표시할 검색어 <a> : 하이퍼링크를 달아서 해당 위치로 연결할 때 사용. -> 기초학습내용으로
  • target 이후 부여하는 속성에 따라 다르게 연결함.
    • _blank : 새탭으로 띄움 -> 네이버
    • _self : 현재에 띄움 -> 네이버
    • _parent : 웹 브라우저의 부모창 -> 네이버
    • _top : 웹 브라우저 전체 영역 -> 네이버
    • 브라우저의 영역이 top-parent-self 와 같이 구분되어 있지 않을 경우에, parent와 top은 self일 때 처럼 동작한다.

  • <th> / <th> : table head의 약자로, 표의제목을 만듬 (기본값은 굵은 글씨체, 중앙정렬)
  • <tr> / <tr> : table row 의 약자로, 가로줄을 만든느 역할 (기본값은 보통 글씨체에, 왼쪽정렬)
  • <td> / <td> : table data 의 약자로, 셀을 만든느 역할 (기본값은 보통 글씨체에, 왼쪽정렬)
  • 속성 colspan=2 : 원하는 만큼의 가로줄을 만들고 병합함.
    가로줄을 만들고 합치기 때문에 추가적으로 칸을 만들필요 없음.
  • 속성 rowspan=2 : 원하는 만큼의 세로줄을 만들고 병합함.
    세로줄을 만들고 합치기 때문에 추가적으로 칸을 만들필요 없음.
  • 속성 colspan:2 rowspan=2 : 2가지 속성을 같이 사용해서, 가로/세로 칸을 합침.
  • <caption> ~ </caption> : 테이블 캡션을 달아주는 태크

  • 회원가입 등에서 입력 양식 만들 때 사용.

    • get : url 뒷부분에 파라미터를 추가해서 전송. 어떤 데이터가 전송하는지 노출되어 보안에 취약함. 노출이 상관없는 데이터 전송시 사용. 2048자의 글자 제한 존재
    • post : http request 헤더에 파라미터를 추가하여 데이터를 전송함. 어떤 데이터를 전송하는지 보이지 않아 보안성이 높음. 로그인 등에 사용. 글자수 제한 없음
