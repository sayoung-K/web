CSS 구성 요소

  • 선택자(selector): 스타일을 적용할 대상 지정
  • 속성명(property): 속성의 이름
  • 속성값(value): 속성에 적용할 값
  • 선택자 {속성명: 속성값;} 과 같이 사용하여 스타일 적용

CSS 적용 방법

  • 인라인 스타일: html 요소의 여는 태그에 style 속성으로 지정
  • 내부 스타일: 같은 html 문서 내부에 <style> 태그를 사용하여 지정
  • 외부 스타일: html 문서와는 개의 파일로 style로 지정

선택자 조합

종류 설명
A B 선택자 B가 포함된 선택자 A
A > B 선택자 A의 직계 자손인 선택자 B 선택
A + B A 선택자 바로 다음의 B 선택자 선택
A ~ B A 선택자 다음에 인접해 있는 모든 선택자 B 선택
A, b 선택자 A와 선택자 B 선택
